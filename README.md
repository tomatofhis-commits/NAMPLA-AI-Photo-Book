# NAMPLA-AI Yukata Ladys

.NET MAUI で開発された、美しいAI生成画像（浴衣美女）の報酬アルバム機能を備えたナンプレ（数独）ゲームアプリです。

---

## 🌟 主な機能・特徴

1. **高品質なナンプレパズル**
   - 3つの難易度（Easy, Medium, Hard）から選択して遊ぶことができます。
   - **解答の一意性保証**: 数学的なバックトラッキング・ソルバーを用いて「答えが常に1つだけ」になるよう問題を動的生成しており、不完全な問題は発生しません。

2. **ご褒美アルバム機能**
   - パズルをクリアするたびに、ゲーム背景に使用された美しい浴衣女性の画像（全449枚）がアンロックされ、アルバムに保存されていきます。
   - 未アンロックの画像が優先的にゲーム背景として出現するため、スムーズにコンプリートを目指せます。
   - すべての画像をアンロックした後は、自動的にランダムで写真が選ばれ、アルバム完成お祝いメッセージに切り替わります。

3. **アルバムの超高速表示と高速シーク**
   - **物理サムネイル最適化**: 450枚近くの大量画像があるアルバム画面でも一瞬で開き、ヌルヌルとスクロールできるように、あらかじめ `200px` 幅のサムネイル画像を同梱して表示に利用しています（拡大プレビュー時は高解像度オリジナルを表示）。
   - **双方向同期シークバー（スライダー）**: アルバム画面の最下部にあるスライダーを操作することで、449枚の任意の画像位置に一瞬でジャンプできます。また、手動でスクロールした際にもスライダーのつまみと現在位置（例: `No. 120 / 449`）が自動で追従します。

4. **安全な Google AdMob バナー広告の統合**
   - タイトル画面の最下部、およびゲームプレイ画面の最上部にバナー広告が表示されます。
   - **デバッグ/本番の自動切り替え**: 開発中の自己クリックによるアカウント停止（BAN）を防ぐため、Debugビルド時はGoogleのテスト用広告、Releaseビルド時のみ本番用の広告が配信されるセーフティ機能を搭載しています。
   - **クロスプラットフォーム配慮**: Windowsビルド時のアセンブリ競合を避けるため、XAMLではなくコードビハインドでプラットフォーム依存コンパイルを用いて動的に広告ビューを追加しています。

## 🛠 使用技術

- **フレームワーク**: .NET 8.0 / .NET MAUI
- **開発言語**: C# 12 / XAML
- **対応プラットフォーム**: Android (API 23以上), Windows (10.0.19041.0以上)
- **使用プラグイン**: `Plugin.MauiMTAdmob` (バージョン 2.0.1.1)
